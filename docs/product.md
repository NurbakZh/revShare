# Product

## Что такое RevShare

RevShare — платформа на Solana которая позволяет малому бизнесу привлекать финансирование через токенизацию доли выручки. Бизнес выпускает токены и продаёт их комьюнити. Инвесторы автоматически получают процент от ежемесячной выручки через смарт-контракт.

**Токен RevShare ≠ акция.** Это право на долю выручки (revenue participation token), не право собственности на бизнес.

---

## Для кого

### Бизнес
Кофейни, рестораны, локальные магазины — любой малый бизнес которому нужно финансирование но нет доступа к банковским кредитам или венчурным инвестициям.

**Что получает:**
- Финансирование без банков и залогов
- Комьюнити лояльных инвесторов которые заинтересованы в успехе бизнеса
- Прозрачную репутацию on-chain — история выплат видна всем

### Инвестор
Обычный человек который хочет инвестировать в локальный бизнес которому доверяет — любимую кофейню, ресторан, магазин.

**Что получает:**
- Доступ к инвестициям от любой суммы (цена одного токена)
- Автоматические ежемесячные выплаты без посредников
- Возможность выйти из позиции через маркетплейс в любой момент

---

## Как это работает

### Флоу бизнеса

```
1. Регистрация
   Бизнес создаёт лот: количество токенов, цена, % выручки инвесторам
   Вносит залог SOL (минимум 30% от суммы привлечения)

2. Продажа токенов
   Токены появляются в каталоге
   Инвесторы покупают — SOL поступает в vault контракта
   После продажи — бизнес получает первые 40% средств

3. Ежемесячные выплаты
   Каждый месяц C# оракул читает выручку из кассы
   Смарт-контракт автоматически распределяет долю держателям токенов
   После первой выплаты — бизнес получает ещё 30%

4. Третий транш
   Когда средняя выручка достигает KPI — бизнес получает последние 30%
```

### Флоу инвестора

```
1. Выбирает бизнес в каталоге
   Видит историю выручки, APY, количество оставшихся токенов

2. Покупает токены
   Подтверждает транзакцию в Phantom
   Токены появляются в кошельке

3. Ежемесячно получает выплаты
   Заходит в dashboard → нажимает Claim
   SOL зачисляется на кошелёк автоматически

4. Хочет выйти из позиции
   Выставляет токены на маркетплейс
   Другой инвестор покупает — сделка происходит on-chain
```

## Защита инвесторов

**Залог бизнеса** — при регистрации бизнес вносит залог SOL. Если бизнес перестаёт платить, залог автоматически идёт держателям токенов.

**Разблокировка 40/30/30** — привлечённые средства выдаются бизнесу поэтапно только при выполнении условий. Экономически невыгодно скамить на малых суммах при залоге в 30%.

**Ранговая система** — новые бизнесы могут привлечь только $500. Лимит растёт автоматически по on-chain истории платежей. Нет смысла скамить на $500 при залоге $150.

**Прозрачность** — все транзакции on-chain, каждый может проверить историю выплат на Solana Explorer.

## Конкурентное преимущество

На Solana нет прямых аналогов revenue sharing платформы для малого бизнеса. Существующие RWA проекты фокусируются на недвижимости и облигациях с высоким порогом входа.

RevShare закрывает сразу несколько пунктов из ТЗ хакатона:
- Токенизация реального актива (выручка бизнеса)
- Proof-of-asset через C# оракул с кассовым API
- Фракционное владение (любое количество токенов)
- Встроенный маркетплейс для торговли токенизированными активами
- Механизм распределения дохода без посредников
